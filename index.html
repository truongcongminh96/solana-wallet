Checklist

A couple of things to verify before you test the app.

1. Install a wallet extension

Phantom is the fastest route for this challenge, but any wallet that implements the Wallet Standard works.

2. Switch the wallet to Devnet

If your extension is on mainnet, the displayed balance will not match the devnet RPC response.

3. Fund the wallet with test SOL

Use the Solana faucet so the UI can show a non-zero balance after you connect.

Production mindset: the wallet owns the private key, your app only asks for permission and reads the public account state.
